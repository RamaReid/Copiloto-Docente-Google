- Expected approach: controlled manual port per file, not blind git apply.

6) Required constraints for Studio work
- Do not reintroduce legacy provider SDKs.
- Do not require external backend credentials for Studio validation.
- Keep all new features compatible with studio/mock runtime.
- Preserve existing routes and core UX flow.

6.1) Legacy terms handling (important)

- Treat those mentions as legacy context only.
- Do not infer required dependencies from those legacy strings.
